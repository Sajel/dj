#### Now you can insert an image

Correct image format to insert are:
`![alt text](url)`

***url*** - should be in following format */help/widget/* **`widgetType`** */img/* **`imageName`**

For instance *`![Test image](/help/widget/htmlwidget/img/img1.jpg)`*
